# Stock Portfolio Tracker

This project is a simple Python-based stock portfolio tracker.

## Description
The program allows users to enter stock names and quantities. It uses a predefined dictionary of stock prices to calculate the total investment value. The result can be displayed on the screen and optionally saved to a file.

## Features
- User input for stock names and quantities
- Uses a dictionary to store stock prices
- Calculates total investment value
- Basic file handling support (CSV/TXT)

## Technologies Used
- Python

## How to Run
1. Open the project folder in VS Code
2. Open the terminal
3. Run the command:
   ```bash
   python stock_tracker.py
